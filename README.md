# 📰 ReportersHub

ReportersHub is a full-stack news platform where community reporters publish articles and users discover news through ratings, engagement, and a modern feed experience.

The platform focuses on community-driven journalism, allowing users to follow reporters, read verified articles, and interact through likes, comments, and views.

# 🚀 Features
👤 User System

User registration & login

Role based access

Reporter request system

Reporter profiles with ratings

📰 Article System

Publish news articles

Upload images/videos

Article moderation by admin

Public article listing

⭐ Engagement Features

❤️ Like articles

💬 Comment on articles

👁 View count tracking

🔖 Save articles

🔎 Explore Page

Users can discover articles through:

Search bar

Category filters

Trending section

Infinite scrolling

📱 Vertical News Feed

A TikTok-style vertical scrolling news feed that allows users to quickly browse articles.

Inspired by modern platforms like:

TikTok

Instagram Reels

🏆 Reporter Ranking

Reporters are ranked based on:

Average rating

Number of ratings

Community feedback

🛠 Admin Panel

Admin can:

Approve or reject reporter requests

Moderate articles

Manage platform content

# ⚙️ Installation & Setup
1️⃣ Clone Repository
git clone https://github.com/yourusername/reportershub.git
2️⃣ Backend Setup (Spring Boot)

Navigate to backend folder.

cd backend

Run the application:

mvn spring-boot:run

Backend runs on:

http://localhost:8080

3️⃣ Frontend Setup (React)

Navigate to frontend folder.

cd frontend

Install dependencies:

npm install

Start frontend:

npm run dev

Frontend runs on:

http://localhost:5173

# 🧠 Key Design Decisions

Role-based system to manage reporters and users

Community rating model for reporter credibility

Explore page with filters for article discovery

Vertical feed UI for fast news consumption

Admin moderation layer to maintain content quality
